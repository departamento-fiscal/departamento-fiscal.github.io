    Como funciona

    • A planilha é validada aqui no navegador (abas, colunas e formatos).
    • Ao publicar, os dados são gravados no repositório do site com o seu token — cada publicação fica registrada no histórico do GitHub.
    • Os usuários da consulta não precisam de login; somente esta página de atualização é restrita.
    • Para trocar a senha desta página, atualize o hash em js/admin.js (instruções no README do repositório).
